# Outdated Docs Archive

This folder contains historical plans, implemented specs, old playtest notes, and reference snapshots. These files are intentionally kept out of the repo root so they do not read like current instructions.

Start with these active docs instead:

- `../STATE.md`
- `../CURRENT_SYSTEMS.md`
- `../ROADMAP.md`
- `../ECONOMY_DESIGN.md`
- `../UI_DESIGN.md`
- `../ASSET_GENERATION.md`
- `../LEVEL_JSON_FORMAT.md`
- `../WEAPON_FRAME_FORMAT.md`
- `../STORY-PREMISE.md`
- `../CREDITS.md`

Archive folders:

- `implemented_specs/`: specs or execution tickets whose major work has landed.
- `old_plans/`: superseded planning docs.
- `old_specs/`: older design sketches that do not describe the current build.
- `snapshots/`: useful historical snapshots of level, enemy, or content state.

Files in this folder can still be useful for context, but current implementation work should be driven by the root docs.
